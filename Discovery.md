**Navigation layer for the active working repository.**

## Repository Role

This is the **active working repository** — lean, connected, and operational. Doctrine and philosophy are developed in the companion repository `Lazarus-Forge-` and refined here into practical implementation.

Divergence between the two repos is a signal, not a problem — it surfaces when doctrine needs updating or practice has drifted from principles. The mechanism: any contributor (human or AI) who encounters a contradiction between repos must log it as a Non-blocking Unknown in `Auditor_Protocols.md` and flag it in the next review cycle. Divergence that goes unlogged is drift. Divergence that gets logged is progress.

---

## What This Repository Is

LazarusForgeV0 is the active working repository for the Lazarus Forge — a salvage-first, adaptive resource recovery system designed to preserve functional value before material reduction.

**Core KPI:** Value recovered per kWh consumed.

**Support_Raft_v0.md** — v0.3
`https://raw.githubusercontent.com/ksarith/LazarusForgeV0/refs/heads/main/Support_Raft_v0.md`
Stationary operational anchor for mobile Leviathan units in open-ocean or high-flow environments. Provides regional power, data relay, physical recovery, and triage processing infrastructure. Uses SWATH hull architecture for wave-zone stability. Features a Sacrificial Shell System that converts biofouling — the primary marine maintenance burden — into a designed output cycle: planned colonization, scheduled shedding, and structured reef substrate contribution. Includes local decision authority framework for satellite blackout periods, conservative energy accounting with explicit infrastructure overhead, Stasis Mode with passive recovery capability preserved, and a self-consuming end-of-region protocol. The Raft's value is measured by what it enables in the units, not what it does directly. Complexity that lives on the Raft stays off the units. Revised after adversarial audit.

Divergence between them is logged, not ignored. See Repository Role above.
